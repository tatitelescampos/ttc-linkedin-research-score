## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues on this repo. External PRs are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

All five canonical labels use their default names (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout — one `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.
